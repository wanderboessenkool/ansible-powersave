laptop_powersave
================

Basic extra powersaving steps for a laptop running Fedora

This role configures powersaving for:
* Generic USB devices
* Genreic PCI devices
* Intel i915 graphic
* Intel snd_hda_intel audio
* Powertop service
* Tuned profiles
